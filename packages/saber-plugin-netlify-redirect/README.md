# saber-plugin-netlify-redirect

Generates `_redirects` file for redirecting '/:page' to '/:page.html' on Netlify.

## Install

```bash
yarn add saber-plugin-netlify-redirect
```

## Usage

In your `saber-config.yml`:

```yml
plugins:
  - resolve: saber-plugin-netlify-redirect
```
